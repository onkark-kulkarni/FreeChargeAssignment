How to use/run the framework
> Clone the git repository
> Open the cloned project in Eclipse
> Open Config.properties file
> Add mobile number and password of your myntra account
> save the config.properties file.
> Login to your myntra account and check you don't have anything in cart(Existing products from cart are not considered in validations)
> From eclipse run the testng.xml file.(right click on testng.xml file and run as testng suite)  or else you can open cmd from Project location and run command mvn test
> After execution go to TestResult foldr and double click on Report.html you will find the latest reports.
